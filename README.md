# AI Resume Analyzer 📄

An automated tool to parse, analyze, and manage resume data efficiently. This project helps in organizing candidate details using SQL and analyzing key information using Python.

## Features ✨
- **Data Parsing:** Extracts key information from resume files.
- **Database Management:** Uses SQL to store and manage patient/candidate records.
- **Automation:** Reduces manual data entry time.

## Technologies Used 🛠️
- **Python:** For data processing and logic.
- **SQL:** For database structure and management.
- **GitHub:** For version control.

## How to Run 🚀
1. Clone the repository: `git clone https://github.com/Surus189/resumeanalyzer.git`
2. Install the necessary libraries (if any).
3. Run the `main.py` file to start the analyzer.
